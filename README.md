# ΏθΥο Installation

Clone repository

```
git clone https://github.com/SubhashPaingoriya-dev/GSAP-Macbook-Air-Pro-Landing-Page
```

Go to project

```
cd GSAP-Macbook-Air-Pro-Landing-Page
```

Install dependencies

```
npm install
```

Run project

```
npm run dev
```
